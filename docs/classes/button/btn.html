BtnElement¶

How to write by SpWML¶

(btn)Button text
+(child block)Block-type buttons use child blocks.

Summary¶

A block in which various buttons can be placed.

Functions (Dart)¶

Child block control¶

Parent Class function. This only has an effect when the button is of block type.

Callback setting¶

Set button callback

void setBtnCallback(void Function()? callback)

Variable（Dart）¶

  • common variable.

  • parent variable

  • [elParams] : Like TextButton class parameters.

Parameters¶

  • common parameters. Parameters related to borders work differently from other elements.

  • parent parameters.

  • type (String). Specify the type of button as one of text, outlined, elevated, icon, block. block is a special button whose child element responds to taps.

  • iconNum (String). Set icon type by codePoint .

  • iconSize (double).

  • iconColor (String). #AARRGGBB, or 2014 Material Design color palettes colors, like red[500] or redAccent[100].

  • splashRadius (double).

  • fgColor (String). The foreground color. Overlay color changes when button is tapped. #AARRGGBB, or 2014 Material Design color palettes colors, like red[500] or redAccent[100].

  • btnBGColor (String). The backgound color. #AARRGGBB, or 2014 Material Design color palettes colors, like red[500] or redAccent[100].

  • color (String). Same btnBGColor.

  • shape (String). The border shape. Specify one of roundRect, stadium, bevel, circle.
