Summary¶

A block that can create radio buttons.

Functions (Dart)¶

Child block control¶

Parent class functions.

Initial value setting¶

Set the initial value.To keep the value when updating the screen, use Hold the value in the parent widget, etc., and call it. It will be in the state where the index th counting from 0 given as an argument is selected.

void setInitialIndex(int? v)

Callback setting¶

When a radio button is selected, its index is passed.

void setCallback(void Function(int? selectedIndex)? callback)

Variable（Dart）¶

  • The common variable.

  • Parent variable.

  • [elParams] : Not only Radio class variables that has variables that have been further adjusted to improve usability.

Parameters¶

  • The common parameters.

  • Parent parameters.

  • iconSize (double). Set size of icon in px.

  • iconColor (String). Color setting. #AARRGGBB notation, or 2014 Material Design color palettes It can be handled with notations such as red[500] and redAccent[100].

  • splashRadius (double). Set the size of the reaction circle by px on when tapped.

  • enableTapLabel (bool). If true, can tap text. The initial value is false.
