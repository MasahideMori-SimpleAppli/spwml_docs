The color system¶

How to write color¶

#AARRGGBB(#RRGGBB) notation, or 2014 Material Design color palettes It can be handled with notations such as red[500] and redAccent[100].

(block, bgColor:#FF9800)
(block, bgColor:orange)
(block, bgColor:orange[500])
