======================================
ColElement
======================================

How to write by SpWML
========================

.. code-block:: none
   
   (col)
   +(child block)
   +(child block)

----

Summary
===============

A block with multiple child blocks.
Each inner child block becomes a row element.

----

----

Parameters
====================

* :ref:`The common<spwml>` parameters.
* :ref:`Parent<multi_child>` variable.
* vAlign (String). Vertical alignment: left, start, center, right, end, top, bottom, spaceAround, spaceEvenly or spaceBetween.
* hAlign (String). Horizontal alignment, which can be left, start, center, right, end, top, bottom or stretch.
